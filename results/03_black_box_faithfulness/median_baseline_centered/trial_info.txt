trial_name: median_baseline_centered
trial_long_name: Faithfulness vs coalition size (median-centered baseline, shifted model G(δ)=F(b+δ))
baseline_identity: b = median(X_train) (x-space), with centered coordinates $\delta=x-b$
model_identity_for_phi: shifted model $G(\delta)=F(b+\delta)$

Definitions:
Err(S) = |F(x_S) - Fhat(x_S)| / (|F(x_S)| + eps(S))
eps(S) = 0.01 * |F(x_S)| + 1e-12
Per-input, per-k mean error E_x(k) = mean_{|S|=k} Err(S)
Across-input mean/std per k computed over E_x(k)
Input-level WinRate(k) = frac_x [E_x_main+int(k) < E_x_main(k)]
Per-input relative improvement RI_x = (E_main - E_main+int) / (E_main + 1e-12)
